== Installation Instructions

- git clone git@github.com:tiagom87/daniel.git
- cd daniel
- bundle install
- rake db:migrate
- rails server

== Notas

- Adições de 3 métodos para o Devise funcionar noutras paginas
- 2 Modals com o Signup e o Login forms copiados dos views do devise
- 2 links no header com data targets para os modals
